# TERRA UNIVERSAL CONVENTION

Repository bridge file for `women-made-central-asia`.

## Why this file exists here

This repository is not a legal donor repository, but it is still part of the
wider Terra ethical line. The convention bridge exists here so that public
research writing remains visibly connected to dignity, safety, and
non-exploitative knowledge work.

## Local interpretation

Inside this repository, the convention line means:

1. women, families, and communities are not treated as extractive content;
2. invisible labor, care work, ritual continuity, and oral memory are treated
   with seriousness and respect;
3. living communities are not reduced to aesthetic material;
4. AI assistance may help with structure, but must not fabricate evidence;
5. public writing must remain safe, sourced, and non-degrading.

## Cross-layer relation

This repository should remain aligned with:

- the public narrative layer;
- the deeper corpus and source archive;
- the wider Terra governance donor layer where applicable.

# Contributing to women-made-central-asia

`women-made-central-asia` is a human-facing research repository inside the wider Terra and FMP ecosystem.

Contributions here should make the project:

- more historically readable;
- more respectful toward women as visible and invisible continuity workers;
- less fragmented;
- less noisy;
- and safer for public interpretation.

## Before contributing

Please read:

- `README.md`
- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- `DOCUMENTATION-STANDARD.md`
- the donor legal layer in `AIUZ-Terra-Codex-EcoSystem/terra-legal`

## Good contribution types

- documentation repair and clarification;
- source cleanup and citation repair;
- mojibake and formatting cleanup;
- public landing page improvement;
- metadata repair for publication and archive surfaces;
- careful enrichment of women, continuity, ritual, textile, memory, and shrine-related materials.

## Avoid

- reducing women to decorative symbolism or token examples;
- mixing archive residue into visible public surfaces without review;
- publishing unfinished machine-generated placeholders as if they were finished human texts;
- committing credentials, secrets, or personal data;
- adding automation clutter to human-facing files.

## Pull request rule

Changes should explain:

- what was changed;
- why it improves the repository for human readers;
- whether the change affects public interpretation, publication, or source integrity.

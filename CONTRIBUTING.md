# CONTRIBUTING

Thank you for contributing to `women-made-central-asia`.

This repository is human-first, source-grounded, and culturally serious.

## Contribution principles

- prefer traceable sources over rhetorical certainty;
- distinguish evidence from interpretation;
- respect named and unnamed historical actors;
- do not flatten women's civilizational work into decorative side notes;
- mark speculative bridges as proposals, not as established repository truth.

## AI rule

AI tools may help with:

- structuring notes;
- language cleanup;
- navigation;
- translation assistance.

AI tools must not:

- invent sources;
- fabricate quotations;
- hallucinate historical facts;
- merge corpus material and public narrative without clear labeling.

## Good contribution types

- clearer README and navigation improvements;
- corrections of dates, names, and transliteration where evidenced;
- source mapping;
- thematic essays grounded in the existing corpus;
- multilingual clarification.

## Submission rule

When changing interpretation-heavy content, explain:

1. what changed;
2. which source or corpus basis supports it;
3. whether it is a correction, expansion, or proposal.

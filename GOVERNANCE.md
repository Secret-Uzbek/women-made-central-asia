# GOVERNANCE

Governance bridge for `women-made-central-asia`.

## Repository role

This repository is the public narrative and interpretation layer of the project.

It is distinct from the structured evidence archive in:

- `women-made-central-asia-corpus`

The public repository should remain readable, curated, and human-first.
The corpus repository should remain the deeper evidence substrate.

## Decision logic

When conflict appears, use this order:

1. protection of people and communities;
2. source integrity;
3. cultural and historical seriousness;
4. public readability;
5. convenience.

## Editorial rule

This repository may contain interpretation, but interpretation must remain visibly grounded.
Do not present synthetic narrative as if it were raw source material.

## Cross-repository rule

If a contribution belongs more properly to the corpus layer, move it there instead of overloading the public repo.

## Ecosystem link

This repository follows the wider Terra governance and ethical line, including child safety, human dignity, and non-exploitative knowledge work.

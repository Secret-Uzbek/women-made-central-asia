# Security Policy

This file describes the repository-specific security boundary for `women-made-central-asia`.

## What belongs here

Report issues that affect:

- repository integrity;
- public-facing pages or documentation;
- accidental exposure of private data, unpublished personal material, or sensitive archive fragments;
- scripts, workflows, or automation files stored in this repository.

## Reporting

Please report security concerns to:

- `a.abdukarimov@fractal-metascience.org`

Do not open public issues for exploitable vulnerabilities, exposed secrets, or sensitive human material.

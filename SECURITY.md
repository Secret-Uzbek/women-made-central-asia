# SECURITY POLICY

Security for `women-made-central-asia` means more than software flaws.
It also includes source integrity, cultural sensitivity, and protection from harmful misuse.

## Report a problem

Contact:

- a.abdukarimov@fractal-metascience.org
- subject prefix: `[SECURITY] women-made-central-asia`

## In scope

- repository integrity problems;
- malicious or misleading alterations;
- sensitive or harmful publication mistakes;
- doxxing, stalking, or exploitative use of living-person material;
- AI-generated fabrication presented as real research.

## Out of scope

- abstract theoretical concerns with no concrete repository impact;
- requests to erase legitimate historical interpretation just because it is uncomfortable.

## Priority rule

Anything affecting:

- children,
- women,
- living communities,
- personal data,

receives immediate priority.
